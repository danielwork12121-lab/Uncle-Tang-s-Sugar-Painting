# Sugar Painting Game

A Chinese traditional sugar painting mini-game built with HTML5 Canvas and Vite.

## Features

- Scene 1: Introduction animation
- Scene 2: Sugar boiling mini-game
- Scene 3: Candy pouring and drawing
- Scene 4-9: Story sequence with circle and dragon candy painting

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Controls

- Press 1-9 keys to jump to specific scenes (dev mode)
- Mouse/touch to draw candy shapes
- Click to progress through story scenes
